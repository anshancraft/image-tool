# 服装电商图片批量整理工具

专为电商卖家打造的一站式图片处理工具，支持Windows/Mac双端运行。

## ✨ 核心功能
- 📝 批量改名、分类归档，支持自定义模板+日期命名
- ✂️ 自定义宽高居中裁剪（适配750x1000/800x800等电商主图尺寸）
- 💧 批量添加文字/Logo水印，可调节位置、字号、透明度
- 📦 批量压缩、格式转换（JPG/PNG/WebP）
- 📊 导出带缩略图的Excel台账，可随单元格自动缩放

## 🚀 使用方法
1.  下载源码或打包好的程序
2.  双击运行，选择图片文件夹
3.  按需设置参数，点击“开始整理”即可

## 📦 运行环境
- Python 3.x
- 依赖库：pillow、openpyxl
